# 메뉴 프로그램

## 개발 기능 목록

- [ ] 코치 이름 입력 기능
    - [ ] ","로 구분되는지 검증
    - [ ] 이름 최소 2글자, 최대 4글자 검증
    - [ ] 코치 최소 2명, 최대 5명 검증

- [ ] 못 먹는 메뉴 입력 기능
    - [ ] ","로 구분되는지 검증
    - [ ] 최소 0개, 최대 2개 검증
    - [ ] 입력된 메뉴가 중복인지 검증

- [ ] 메뉴 검증 기능
    - [ ] 존재하는 메뉴인지 검증

- [ ] 카테고리 추천 기능

- [ ] 카테고리 검증 기능
    - [ ] 추천 이력이 2회 이상인지 검증

- [ ] 메뉴 추천 기능

- [ ] 메뉴 검증 기능
    - [ ] 못 먹는 메뉴인지 검증
    - [ ] 추천 이력이 있는 메뉴인지 검증

- [ ] 메뉴 추천 출력 기능

- [ ] 추천 이력 저장 기능
